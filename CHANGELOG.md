## [1.0.0] - 2025-11-18

### Added
- Initial release
- Automatic documentation generation for MCP servers
- Support for extracting tools, resources, and prompts from MCP server classes
- Beautiful, responsive UI built with Tailwind CSS
- Configurable installation commands for different platforms
- Publishable configuration and views
- Comprehensive error handling and validation
- Basic test suite

### Features
- Automatic route registration at `/docs/mcp` (configurable)
- Customizable server URLs and installation commands
- Support for authentication messages
- Support for custom support/help messages
- View customization through publishing
